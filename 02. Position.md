# Position

Digunakan untuk mengatur posisi element. Jadi bisa lebih spesifik ke element tersebut.

- `static` element static by default (document flow)
- `relative`: static, `left dan right` menggeser posisi element dari posisi seharusnya, element lain akan mengakui posisi awalnya.
- `fixed` menempatkan elemen relatif terhadap viewport, bukan terhadap parent. Artinya, top, left, right, dan bottom diukur dari tepi jendela browser yang terlihat saat ini. Tetapi, jika tidak diatur elemen tetap berada di posisi aslinya (secara visual) namun tetap out of flow. Saat halaman di-scroll, elemen dengan position: fixed tetap di tempat (tidak ikut bergerak dengan konten).

⚠️ Pengecualian modern: jika elemen `fixed` berada di dalam parent yang memiliki transform, filter, perspective, will-change: transform, atau contain: paint, maka posisi `fixed`.

- `absolute` keluar dari document flow completely, jika left/right tidak diatur akan di posisi document flow, jika diatur akan di posisi parent terdekat yang memiliki position (absolute, relative, sticky, fixed)

sticky: seperti relative, tetapi ketika keluar frame, akan stacking

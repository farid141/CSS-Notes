# Display

Untuk mengatur bagaimana elemen dirender dalam flow dokumen, dan bagaimana anak-anaknya diatur. Dari pernyataan itu, sebenarnya display memiliki dua vaule `display: <outer-display> <inner-display>;`

Tapi kebanyakan menggunakan satu property

```css
{
display: block; /* = block flow */
display: flex;  /* = block flex */
}
```

## Outer display

Menentukan bagaimana elemen berperilaku terhadap elemen lain di sekitarnya (apakah blok baru, inline, dll).

Nilai umum:

- block
- inline
- run-in (jarang dipakai)
- list-item (untuk `<li>`)

## Inner display

Menentukan bagaimana isi elemen diatur di dalamnya (flow normal, flex, grid, table).

Nilai umum:

- flow â†’ default (menggunakan normal flow)
- flex
- grid
- ruby
- table

## Default display

Beberapa element memiliki display default:

- span: inline
- div: block

## Nilai Umum Display

- inline-block: secara default inline, tapi bisa kita kasih width/height

jika width/height diset kurang dari konten, secara default akan berukuran konten dan menggeser element tetangga
